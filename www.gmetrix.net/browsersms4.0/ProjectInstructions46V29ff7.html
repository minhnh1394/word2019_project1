Instructions:

In this Word Project, you will be creating a section of a plant book. The document that is open is your starting point. You will not need to open any other document to complete this project. All resources that are needed to create this document are located in the GMetrixTemplates folder in your Documents folder. Follow the tasks below to complete the project. You can use the checkboxes to keep track of your progress. In Training mode, you can click the lightbulb to see step by step help for the instruction. Note: Accept default values unless otherwise instructed.

Note: You should save your progress frequently during the practice test.
Using the Save button below will save a record of your test on the GMetrix servers, but your document progress will be saved on your local computer.
To resume the test from a different location, you will need to save a copy of your document manually.

Entire document

Page Border

Add a page border to a document.

Entire document

  • Accept all other defaults

Modify Style

Modify the color of the Title style.

First paragraph, the text Flowers!

  • Gold Accent 3, Darker 25%
1. The text Flowers! should still be selected. If not, select it again.
2. In the HOME tab, Styles group, select the More down-arrow to open the Styles gallery.
3. Right-click Title. (Hint: Alternately you can click the dialog box launcher in the lower-right corner of the Styles group to open the Styles Task Pane and click the drop-down for Title.)
4. Select Modify.
5. In the Modify Style dialogue window, under the Formatting section, select the Font Color drop-down.
6. Select Gold Accent 3, Darker 25%
7. Click OK.

Bulleted List

  • First Level: Location, Color of the blooms, Usefullness, Allergens

12.
Macro

Create a macro.

Store the macro in the current document

  • Name the macro, "Cooper12"
  • Actions: Cooper Black font, 12 pt
1. In the VIEW tab, Macros group, select the Macros drop-down.
2. Select Record Macro.
3. In the field Macro name: type "Cooper12".
4. Select the drop-down for Store macro in: and select Flowers (document).
Note: If you have saved and resumed your test, the document name will have changed. Select the option that ends with the word (document).
5. Accept all other defaults and click OK.
6. In the HOME tab, Font group, select the Font drop-down. Select Cooper Black.
7. In the HOME tab, Font group, select the Size drop-down and select 12.
8. In the VIEW tab, Macros group, select the drop-down for Macros.
9. Select Stop Recording.

  • Banded quote
  • Position: Bottom Right with Square Text Wrapping

SmartArt

Insert a SmartArt diagram into the document.

On the last page, in the second column.

  • SmartArt Basic Chevron Process
  • Style: 3-D Polished
  • Shape 1 Text: "Plant"
  • Shape 2 Text: "Feed and Water"
  • Shape 3 Text: "Bloom"
  • Position: Top Right with Square Text Wrapping

9. For text in the third shape, type "Bloom".
10. In the SMARTART TOOLS/FORMAT tab, Arrange group, select the Position drop-down.

30.
Document Options

Change document options.

General options

  • Disable: Enable Live Preview
1. In the FILE tab, click Options.
2. In the General tab, click next to Enable Live Preview, so that there is not a check mark next to it.
3. Click OK.

Proofing options

  • Enable: Check spelling as you type
1. In the FILE tab, select Options.
2. In the Proofing tab, under When correcting grammar and spelling in Word click so there is a check mark next to Check spelling as you type.
3. Click OK.

Advanced Options

  • Keep track of formatting
  • Print pages in reverse order
1. In the FILE tab, select Options.
2. In the Advanced tab, under the Editing options section, click to select the box next to Keep track of formatting.
3. Under the Print section, click to select the box next to Print in reverse order.
4. Click OK.

Note: If you do not close the Word Options menu before grading your test it will cause an issue with scoring.
